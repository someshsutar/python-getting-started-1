### What is Python?

**Python** is a high-level, interpreted, general-purpose programming language created by Guido van Rossum in 1991.

It is designed to be:

* Easy to read and write
* Beginner-friendly
* Cross-platform (Windows, Linux, macOS)
* Suitable for web development, automation, data analysis, AI, machine learning, and more

Example:

```python
print("Hello, World!")
```

Python's syntax is simple and resembles plain English, making it easier to learn than many other programming languages.

## Why is Python so popular for AI and Machine Learning?

Python is not the only language that can be used for AI and ML, but it has become the industry standard for several reasons.

### 1. Simple and Readable Syntax

AI and ML involve complex mathematical concepts. Python allows developers and researchers to focus on algorithms rather than language complexity.

Example:

```python
if accuracy > 0.95:
    print("Model is ready")
```

This simplicity speeds up development and experimentation.

Python has a vast collection of libraries specifically designed for AI and ML:
