In Python, **running a script** and using the **REPL** are two different ways of executing Python code.

## 1. REPL (Read-Eval-Print Loop)

REPL is Python's interactive shell.

The name comes from the cycle:

```text
Read  -> Read your input
Eval  -> Execute it
Print -> Show the result
Loop  -> Wait for next input
```

### Starting REPL

Open a terminal and type:

```bash
python
```

or

```bash
python3
```

You'll see:

```python
Python 3.13.0
>>>
```

Now you can execute code line by line:

```python
>>> x = 10
>>> y = 20
>>> x + y
30
>>>
```

### Characteristics

✅ Immediate feedback

✅ Great for learning

✅ Useful for testing small snippets

✅ Variables remain in memory during the session

❌ Code is lost when you exit

❌ Not suitable for large applications

---

## 2. Running a Script

A script is a Python file (`.py`) containing code.

Example: `hello.py`

```python
name = "Someshwar"

print(f"Hello {name}")
```

Run it:

```bash
python hello.py
```

Output:

```text
Hello Someshwar
```

### Characteristics

✅ Code is saved in a file

✅ Can contain thousands of lines

✅ Version control friendly (Git)

✅ Suitable for applications and automation

❌ Requires editing and re-running to test changes

---

## 3. Execution Difference

### REPL

When you type:

```python
>>> x = 10
```

Python immediately:

```text
Input
  ↓
Compile to Bytecode
  ↓
Execute
  ↓
Display Result
```

One statement at a time.

---

### Script

For:

```python
x = 10
y = 20

print(x + y)
```

Python:

```text
Read Entire File
      ↓
Compile
      ↓
Execute from top to bottom
      ↓
Exit
```

---

## 4. Memory Behavior

### REPL

Variables stay alive until you exit:

```python
>>> x = 10
>>> x
10
>>> x + 5
15
```

Exit:

```python
>>> exit()
```

Everything is gone.

---

### Script

```python
x = 10
print(x)
```

After execution:

```text
Program ends
Memory released
```

Variables disappear automatically.

---

## 5. Error Handling Example

### REPL

```python
>>> 10 / 0
ZeroDivisionError
```

You can immediately try again:

```python
>>> 10 / 2
5.0
```

---

### Script

```python
print(10 / 0)
print("Done")
```

Output:

```text
ZeroDivisionError
```

Program stops unless exception handling is used.

## 6. Typical Usage

### Use REPL for

* Learning Python
* Experimenting with APIs
* Testing regular expressions
* Verifying logic
* Quick calculations

Example:

```python
>>> import math
>>> math.sqrt(81)
9.0
```

---

### Use Scripts for

* Web applications
* AI/ML projects
* Automation
* Data processing
* Microservices

Example:

```bash
python train_model.py
python app.py
python data_import.py
```
