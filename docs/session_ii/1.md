# Data Types & Variables in Python

Variables and data types are the foundation of Python programming.

Python is **dynamically typed**, meaning you don't explicitly declare the variable type.

---

# 1. Variables in Python

A variable is simply a name that refers to a value.

```python
name = "Someshwar"
age = 40
salary = 100000.50
```

Unlike C#:

```csharp
string name = "Someshwar";
int age = 40;
decimal salary = 100000.50M;
```

Python automatically determines the type.

---

# 2. Checking Variable Types

Use the `type()` function:

```python
name = "Someshwar"
age = 40

print(type(name))
print(type(age))
```

Output:

```text
<class 'str'>
<class 'int'>
```

---

# 3. Python Primitive Data Types

## Integer (int)

Whole numbers.

```python
count = 100
temperature = -5
```

Examples:

```python
print(type(count))
```

Output:

```text
<class 'int'>
```

---

## Float (float)

Decimal numbers.

```python
price = 99.99
pi = 3.14159
```

Example:

```python
print(type(price))
```

Output:

```text
<class 'float'>
```

---

## String (str)

Text values.

```python
first_name = "Someshwar"
city = 'Nagpur'
```

Examples:

```python
print(first_name)
print(len(first_name))
```

Output:

```text
Someshwar
9
```

---

## Boolean (bool)

Represents True or False.

```python
is_active = True
is_admin = False
```

Example:

```python
print(type(is_active))
```

Output:

```text
<class 'bool'>
```

---

## NoneType

Equivalent to `null` in C#.

```python
customer = None
```

Example:

```python
print(type(customer))
```

Output:

```text
<class 'NoneType'>
```

---

# 4. Dynamic Typing

A variable can change type during execution.

```python
value = 10

print(type(value))

value = "Hello"

print(type(value))
```

Output:

```text
<class 'int'>
<class 'str'>
```

This is legal in Python.

In C#, this would require:

```csharp
object value = 10;
value = "Hello";
```

---

# 5. Multiple Assignment

Python allows assigning multiple variables at once.

```python
name, age, city = "Someshwar", 40, "Nagpur"
```

Example:

```python
print(name)
print(age)
print(city)
```

---

# 6. Variable Naming Rules

Valid:

```python
first_name = "Someshwar"
_age = 40
salary2025 = 100000
```

Invalid:

```python
2name = "John"      # Starts with digit
first-name = "John" # Hyphen not allowed
```

---

# 7. Collection Data Types

These are heavily used in AI, ML, and data processing.

## List

Ordered and mutable.

```python
fruits = ["Apple", "Mango", "Orange"]
```

Access elements:

```python
print(fruits[0])
```

Output:

```text
Apple
```

Add element:

```python
fruits.append("Banana")
```

Equivalent C#:

```csharp
List<string> fruits = new();
```

---

## Tuple

Ordered and immutable.

```python
coordinates = (10, 20)
```

Cannot modify:

```python
coordinates[0] = 5
```

Produces:

```text
TypeError
```

Equivalent C#:

```csharp
(int x, int y) coordinates = (10, 20);
```

---

## Set

Stores unique values.

```python
numbers = {1, 2, 3, 4}
```

Duplicates removed automatically:

```python
numbers = {1, 2, 2, 3}
```

Result:

```text
{1, 2, 3}
```

Equivalent C#:

```csharp
HashSet<int>
```

---

## Dictionary

Key-value pairs.

```python
employee = {
    "id": 101,
    "name": "Someshwar",
    "city": "Nagpur"
}
```

Access value:

```python
print(employee["name"])
```

Output:

```text
Someshwar
```

Equivalent C#:

```csharp
Dictionary<string, object>
```

---

# 8. Type Conversion

## Integer to String

```python
age = 40

age_text = str(age)
```

---

## String to Integer

```python
age = int("40")
```

---

## Integer to Float

```python
price = float(100)
```

---


```text

```

(decimal part removed)

# 9. String Formatting

### f-Strings (Recommended)

```python
name = "Someshwar"
age = 40

print(f"My name is {name} and I am {age} years old")
```

Output:

```text
My name is Someshwar and I am 40 years old
```

Similar to C#:

```csharp
$"My name is {name} and I am {age} years old"
```

---

# 10. Mutable vs Immutable

## Immutable Types

Cannot be changed after creation.

```python
int
float
bool
str
tuple
```

Example:

```python
name = "Someshwar"

name = name + " Sutar"
```

A new string object is created.

---

## Mutable Types

Can be modified.

```python
list
dict
set
```

Example:

```python
items = [1, 2, 3]

items.append(4)
```

Same object is updated.

---

# 11. Everything is an Object

In Python:

```python
x = 10
```

Even an integer is an object:

```python
print(type(x))
```

Output:

```text
<class 'int'>
```

You can inspect it:

```python
print(dir(x))
```

This will display methods and properties available on the integer object.

# Example Combining Everything

```python
employee = {
    "id": 101,
    "name": "Someshwar",
    "skills": ["Python", ".NET", "Angular"],
    "active": True
}

print(f"Employee: {employee['name']}")
print(f"Skills: {employee['skills']}")
```

Output:

```text
Employee: Someshwar
Skills: ['Python', '.NET', 'Angular']
```

For AI and ML development, you'll work heavily with:

* `str` (text processing)
* `list` (datasets, token lists)
* `dict` (JSON, configurations, API responses)
* `float` (model weights, probabilities)
* `None` (missing values)
* Objects from libraries like **NumPy**, **Pandas**, and **PyTorch**.
