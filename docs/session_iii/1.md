# Control Flow in Python

**Control Flow** determines the order in which statements execute in a program.

Just like C# has:

```csharp
if
else
switch
for
while
break
continue
```

Python provides similar constructs with a cleaner syntax.

---

# 1. if Statement

Used to execute code conditionally.

### Syntax

```python
if condition:
    statement
```

Example:

```python
age = 20

if age >= 18:
    print("Eligible to vote")
```

Output:

```text
Eligible to vote
```

Notice the **colon (`:`)** and **indentation**.

Python uses indentation instead of `{}`.

C#:

```csharp
if(age >= 18)
{
    Console.WriteLine("Eligible to vote");
}
```

Python:

```python
if age >= 18:
    print("Eligible to vote")
```

---

# 2. if-else Statement

```python
age = 16

if age >= 18:
    print("Adult")
else:
    print("Minor")
```

Output:

```text
Minor
```

---

# 3. if-elif-else

Equivalent to multiple `else if` conditions.

```python
score = 85

if score >= 90:
    print("Grade A")
elif score >= 75:
    print("Grade B")
elif score >= 60:
    print("Grade C")
else:
    print("Grade D")
```

Output:

```text
Grade B
```

---

# 4. Nested if

```python
age = 25
citizen = True

if age >= 18:
    if citizen:
        print("Eligible to vote")
```

Output:

```text
Eligible to vote
```

---

# 5. Ternary Operator

Compact conditional expression.

```python
age = 20

status = "Adult" if age >= 18 else "Minor"

print(status)
```

Output:

```text
Adult
```

Equivalent C#:

```csharp
var status = age >= 18 ? "Adult" : "Minor";
```

---

# 6. match-case (Python's Switch)

Introduced in Python 3.10.

```python
day = 2

match day:
    case 1:
        print("Monday")
    case 2:
        print("Tuesday")
    case 3:
        print("Wednesday")
    case _:
        print("Invalid Day")
```

Output:

```text
Tuesday
```

Equivalent C#:

```csharp
switch(day)
{
    case 1:
        Console.WriteLine("Monday");
        break;
}
```

# 7. for Loop

Used to iterate over collections.

### Example 1

```python
for i in range(5):
    print(i)
```

Output:

```text
0
1
2
3
4
```

---

### Example 2

```python
skills = ["Python", ".NET", "Angular"]

for skill in skills:
    print(skill)
```

Output:

```text
Python
.NET
Angular
```

---

# 8. range()

Generates a sequence of numbers.

### range(stop)

```python
for i in range(5):
    print(i)
```

Produces:

```text
0 1 2 3 4
```

---

### range(start, stop)

```python
for i in range(1, 6):
    print(i)
```

Produces:

```text
1 2 3 4 5
```

---

### range(start, stop, step)

```python
for i in range(0, 10, 2):
    print(i)
```

Output:

```text
0 2 4 6 8
```

---

# 9. while Loop

Executes while condition is True.

```python
count = 1

while count <= 5:
    print(count)
    count += 1
```

Output:

```text
1
2
3
4
5
```

Equivalent C#:

```csharp
while(count <= 5)
{
    Console.WriteLine(count);
    count++;
}
```

---

# 10. break Statement

Terminates loop immediately.

```python
for i in range(10):
    if i == 5:
        break

    print(i)
```

Output:

```text
0
1
2
3
4
```

---

# 11. continue Statement

Skips current iteration.

```python
for i in range(5):
    if i == 2:
        continue

    print(i)
```

Output:

```text
0
1
3
4
```

---

# 12. pass Statement

Placeholder for future code.

```python
if True:
    pass
```

Useful during development:

```python
def calculate():
    pass
```

Equivalent to:

```csharp
void Calculate()
{
}
```

---

# 13. Loop with else

A unique Python feature.

```python
for i in range(5):
    print(i)
else:
    print("Loop completed")
```

Output:

```text
0
1
2
3
4
Loop completed
```

The `else` executes if the loop finishes normally (not via `break`).

# 14. Enumerate

Gets index and value together.

```python
skills = ["Python", ".NET", "Angular"]

for index, skill in enumerate(skills):
    print(index, skill)
```

Output:

```text
0 Python
1 .NET
2 Angular
```

# 15. Iterating Dictionary

```python
employee = {
    "id": 101,
    "name": "Someshwar",
    "city": "Pune"
}
```

Keys:

```python
for key in employee:
    print(key)
```

Output:

```text
id
name
city
```

---

Key-Value Pairs:

```python
for key, value in employee.items():
    print(key, value)
```

Output:

```text
id 101
name Someshwar
city Pune
```

# 16. List Comprehension (Python Feature)

A concise way to create lists.

Traditional:

```python
numbers = []

for i in range(5):
    numbers.append(i)
```

Pythonic:

```python
numbers = [i for i in range(5)]
```

Result:

```python
[0, 1, 2, 3, 4]
```

---

With condition:

```python
evens = [i for i in range(10) if i % 2 == 0]
```

Output:

```python
[0, 2, 4, 6, 8]
```

This feature is heavily used in AI and data processing.

# Truthy and Falsy Values

Unlike C#, Python treats many values as True or False.

Falsy values:

```python
False
None
0
0.0
''
[]
{}
set()
```

Example:

```python
name = ""

if name:
    print("Valid")
else:
    print("Empty")
```

Output:

```text
Empty
```

# Practical Example

```python
employees = [
    {"name": "Someshwar", "salary": 100000},
    {"name": "John", "salary": 80000},
    {"name": "Mary", "salary": 120000}

```

# Python vs C# Control Flow

| Concept        | Python               | C#                        |
| -------------- | -------------------- | ------------------------- |
| Conditional    | `if`, `elif`, `else` | `if`, `else if`, `else`   |
| Switch         | `match-case`         | `switch`                  |
| Loop           | `for`, `while`       | `for`, `foreach`, `while` |
| Skip Iteration | `continue`           | `continue`                |
| Exit Loop      | `break`              | `break`                   |
| Ternary        | `x if cond else y`   | `cond ? x : y`            |
| Empty Block    | `pass`               | `{}`                      |

## Most Important Control Flow Concepts for AI/ML

As you move into Python for AI and data science, you'll use these constantly:

1. `if / elif / else` – model decisions and validations.
2. `for` loops – processing datasets.
3. `while` loops – training iterations and retries.
4. `break` and `continue` – controlling long-running processes.
5. List comprehensions – transforming data efficiently.
6. Dictionary iteration – handling JSON and API responses.

These form the foundation for working with libraries such as NumPy, Pandas, PyTorch, and TensorFlow.
