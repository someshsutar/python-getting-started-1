# Functions in Python

A **function** is a reusable block of code that performs a specific task.

Functions help you:

* Avoid code duplication
* Improve readability
* Organize code
* Make testing easier

If you're from C#, Python functions are very similar to methods.

---

# 1. Defining a Function

Use the `def` keyword.

```python
def greet():
    print("Hello, World!")
```

Calling the function:

```python
greet()
```

Output:

```text
Hello, World!
```

Equivalent C#:

```csharp
void Greet()
{
    Console.WriteLine("Hello, World!");
}
```

---

# 2. Function with Parameters

```python
def greet(name):
    print(f"Hello, {name}")
```

Call:

```python
greet("Someshwar")
```

Output:

```text
Hello, Someshwar
```

Equivalent C#:

```csharp
void Greet(string name)
{
    Console.WriteLine($"Hello, {name}");
}
```

---

# 3. Function Returning a Value

```python
def add(a, b):
    return a + b
```

Usage:

```python
result = add(10, 20)

print(result)
```

Output:

```text
30
```

Equivalent C#:

```csharp
int Add(int a, int b)
{
    return a + b;
}
```

---

# 4. Type Hints (Recommended)

Python doesn't require types, but you can provide them.

```python
def add(a: int, b: int) -> int:
    return a + b
```

Usage:

```python
result = add(10, 20)
```

Type hints:

* Improve readability
* Help IDEs
* Help static analysis tools

Unlike C#, they are not enforced at runtime.

# 6. Named Arguments

Python allows passing arguments by name.

```python
def create_user(name, age, city):
    print(name, age, city)
```

Call:

```python
create_user(
    city="Pune",
    age=40,
    name="Someshwar"
)
```

Output:

```text
Someshwar 40 Pune
```

Order doesn't matter when using named arguments.

---

# 7. Multiple Return Values

Python can return multiple values.

```python
def calculate(a, b):
    return a + b, a - b
```

Usage:

```python
sum_value, diff_value = calculate(10, 5)

print(sum_value)
print(diff_value)
```

Output:

```text
15
5
```

Equivalent C#:

```csharp
(int sum, int diff) Calculate(int a, int b)
{
    return (a + b, a - b);
}
```

---

# 8. Variable-Length Arguments (*args)

Accepts any number of positional arguments.

```python
def total(*numbers):
    return sum(numbers)
```

Usage:

```python
print(total(10, 20))
print(total(10, 20, 30))
print(total(10, 20, 30, 40))
```

Output:

```text
30
60
100
```

Think of `*args` as:

```csharp
void Method(params int[] values)
{
}
```

---

# 9. Keyword Arguments (**kwargs)

Accepts arbitrary named arguments.

```python
def display_user(**kwargs):
    for key, value in kwargs.items():
        print(key, value)
```

Usage:

```python
display_user(
    name="Someshwar",
    age=40,
    city="Pune"
)
```

Output:

```text
name Someshwar
age 40
city Pune
```

---

# 10. Combining Parameters

```python
def create_employee(name, *skills, **details):
    print("Name:", name)
    print("Skills:", skills)
    print("Details:", details)
```

Usage:

```python
create_employee(
    "Someshwar",
    "Python",
    ".NET",
    "Angular",
    city="Pune",
    experience=17
)
```

Output:

```text
Name: Someshwar
Skills: ('Python', '.NET', 'Angular')
Details: {'city': 'Pune', 'experience': 17}
```

# 11. Lambda Functions

Anonymous one-line functions.

Normal function:

```python
def square(x):
    return x * x
```

Lambda:

```python
square = lambda x: x * x
```

Usage:

```python
print(square(5))
```

Output:

```text
25
```

Equivalent C#:

```csharp
Func<int, int> square = x => x * x;
```

# 14. Closures

A nested function can access variables from its enclosing scope.

```python
def multiplier(factor):

    def multiply(number):
        return number * factor

    return multiply
```

Usage:

```python
double = multiplier(2)

print(double(10))
```

Output:

```text
20
```

Closures are widely used in decorators and AI pipelines.

---

# 15. Recursion

A function calling itself.

```python
def factorial(n):

    if n == 1:
        return 1

    return n * factorial(n - 1)
```

Usage:

```python
print(factorial(5))
```

Output:

```text
120
```

# 17. Scope (Local vs Global)

### Global Variable

```python
company = "ABC Corp"
```

### Local Variable

```python
def show_company():
    company_name = "XYZ Corp"

    print(company_name)
```

`company_name` exists only inside the function.

# 18. Practical Example

```python
def calculate_bonus(salary, experience):

    if experience >= 10:
        bonus_percentage = 20
    elif experience >= 5:
        bonus_percentage = 10
    else:
        bonus_percentage = 5

    bonus = salary * bonus_percentage / 100

    return bonus


salary = 120000
experience = 17

bonus = calculate_bonus(salary, experience)

print(f"Bonus: {bonus}")
```

Output:

```text
Bonus: 24000.0
```

# Functions Commonly Used in AI/ML

You will frequently write functions for:

### Data Cleaning

```python
def clean_text(text):
    return text.lower().strip()
```

### Feature Engineering

```python
def calculate_age_group(age):
    return "Adult" if age >= 18 else "Minor"
```

### Model Evaluation

```python
def accuracy(correct, total):
    return correct / total
```

### API Processing

```python
def extract_response(response):
    return response["answer"]
```

# Python vs C#

| Python           | C#                               |
| ---------------- | -------------------------------- |
| `def`            | Method                           |
| `return`         | `return`                         |
| `*args`          | `params`                         |
| `**kwargs`       | Dictionary / optional parameters |
| `lambda`         | Lambda expression                |
| Multiple returns | Tuple return                     |
| Type hints       | Strong typing                    |

## Example for Your Background (.NET + AI)

```python
def get_order_summary(order_id: int) -> dict:
    return {
        "order_id": order_id,
        "status": "Completed",
        "amount": 1500
    }


order = get_order_summary(1001)

print(order)
```

Output:

```python
{
    'order_id': 1001,
    'status': 'Completed',
```
